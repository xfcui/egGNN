## Usage

Clone this repository by:
```bash
git clone https://github.com/xfcui/egGNN.git
```

Install packages by:
```bash
pip install -r requirements.txt
```

Run:
```bash
./run_ligpro.py --ligand SMILES --protein **.pdb --gpu GPU_NUM
```
